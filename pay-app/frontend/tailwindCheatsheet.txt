| Class               | Meaning           | What It Does                                                                                                        |
| ------------------- | ----------------- | ------------------------------------------------------------------------------------------------------------------- |
| **`sticky`**        | CSS position mode | Makes the header “stick” to the top of the screen when you scroll. It’s like `position: sticky;`.                   |
| **`top-0`**         | Offset            | Keeps the sticky element pinned to the **very top** (`top: 0px`).                                                   |
| **`z-10`**          | Layer order       | Sets `z-index: 10;`, meaning this element stays **above** others that have lower z-index (useful when overlapping). |
| **`border-b`**      | Border            | Adds a **bottom border**. This visually separates the header from the rest of the page.                             |
| **`bg-white/70`**   | Background color  | Sets a **white background** with **70% opacity** (slightly transparent).                                            |
| **`backdrop-blur`** | Blur effect       | Applies a **blur** to whatever is behind it — gives a frosted glass look when transparency is present.              |




FLEX
| Property          | What it controls                                  | Example                             |
| ----------------- | ------------------------------------------------- | ----------------------------------- |
| `display: flex`   | Turns on flexbox                                  | `class="flex"`                      |
| `flex-direction`  | Sets row (`x-axis`) or column (`y-axis`) layout   | `flex-row` (default), `flex-col`    |
| `justify-content` | Controls how items spread **along the main axis** | `justify-between`, `justify-center` |
| `align-items`     | Controls alignment **on the cross axis**          | `items-center`, `items-start`       |
| `gap`             | Adds spacing between children                     | `gap-4`                             |


| CSS                              | Tailwind class    |
| -------------------------------- | ----------------- |
| `display: flex`                  | `flex`            |
| `flex-direction: row`            | `flex-row`        |
| `flex-direction: column`         | `flex-col`        |
| `justify-content: center`        | `justify-center`  |
| `justify-content: space-between` | `justify-between` |
| `align-items: center`            | `items-center`    |
| `align-items: start`             | `items-start`     |
| `gap: 1rem`                      | `gap-4`           |





| Class                | Meaning                     | What It Does                                                                                                                                                               |
| -------------------- | --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`inline-flex`**    | Display type                | Makes this element behave like text (`inline`), but with Flexbox layout. This means it can appear next to other text or icons. Also enables flex alignment inside the div. |
| **`h-10`**           | Height                      | Sets height to `2.5rem` (≈ 40px).                                                                                                                                          |
| **`w-10`**           | Width                       | Sets width to `2.5rem` (≈ 40px). This, combined with `h-10`, makes it a square.                                                                                            |
| **`items-center`**   | Flex alignment (vertical)   | Centers the content vertically inside the flex container.                                                                                                                  |
| **`justify-center`** | Flex alignment (horizontal) | Centers the content horizontally. Together with `items-center`, the initials go exactly in the middle.                                                                     |
| **`rounded-full`**   | Border radius               | Makes the square perfectly circular. Without this, it would just be a gray square.                                                                                         |
| **`bg-gray-200`**    | Background color            | Fills the circle with a **light gray** color.                                                                                                                              |
| **`text-gray-700`**  | Text color                  | Makes the initials a **dark gray**, giving contrast against the light background.                                                                                          |



| Class            | Meaning          | What It Does                                                                                               |
| ---------------- | ---------------- | ---------------------------------------------------------------------------------------------------------- |
| **`rounded-xl`** | Border radius    | Smoothly rounds the corners of the box (`xl` means extra large radius).                                    |
| **`border`**     | Border           | Adds a thin, light gray border around the card (default border color).                                     |
| **`bg-white`**   | Background color | Makes the card’s background pure white.                                                                    |
| **`p-4`**        | Padding          | Adds `1rem` (≈16px) padding inside the box, so the text doesn’t touch the edges.                           |
| **`shadow-sm`**  | Box shadow       | Adds a **small soft shadow**, making the box look slightly raised above the background (more “card-like”). |



| Class               | Meaning   | What It Does                                             |
| ------------------- | --------- | -------------------------------------------------------- |
| **`text-sm`**       | Text size | Makes the text small (`≈14px`).                          |
| **`text-gray-500`** | Color     | Medium gray text color — subtle, not attention-grabbing. |
